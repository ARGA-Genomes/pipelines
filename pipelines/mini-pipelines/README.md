# Mini pipelines

Independed GBIF pipelines for ingestion biodiversity data, can be started ony as a independed jar, uses embeded Spark to run pipeline

 - [DwcaPipeline.java](https://github.com/gbif/artery/tree/master/pipelines/mini-pipelines/src/main/java/org/gbif/pipelines/minipipelines/DwcaPipeline.java)
 - [DwcaPipelineOptions.java](https://github.com/gbif/artery/tree/master/pipelines/mini-pipelines/src/main/java/org/gbif/pipelines/minipipelines/DwcaPipelineOptions.java)
